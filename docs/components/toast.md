<toast></toast>

|属性|作用|值|
|---|---|---|
| autoClose  | 是否自动关闭 | true/false |
| autoCloseDelay  | 自动关闭间隔时间（秒） | number |
| showClose  | 是否显示关闭图标 | true/false |
| position  | toast显示的位置 | top/middle/bottom|
| closeCallBack  | 关闭执行的回调 | function |
| useHtml  | 显示内容是否使用html |  true/false  |
